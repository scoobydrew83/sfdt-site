# Configuration reference — new keys (Phase 1 + 2)

Merge these into the existing `.sfdt/config.json` reference. All keys are
validated; channel/SMTP secrets are referenced by env-var **name**, never inline.

## `notifications`

```jsonc
{
  "notifications": {
    "enabled": false,                 // master switch for the channels[] system
    "summary": { "enabled": false },  // send AI executive summaries for snapshots
    "channels": [
      // type: slack | teams | email | webhook
      // common: name, severityThreshold (ok|warn|error|fail), events[]
      // slack/teams: webhookUrl | webhookUrlEnv
      // webhook: url | urlEnv, format (json|loki), headers{}
      // email: from, to[], smtp{ hostEnv, portEnv, userEnv, passwordEnv, secure }
    ],
    // legacy single-Slack shape (still honored when features.notifications=true):
    "slack": { "webhookUrl": "", "webhookUrlEnv": "" }
  }
}
```
